**🎓 사업부 신입사원 온보딩 AI 강의**도 맡아, 신규 입사자에게 **AI 업무 적용 방안·실전 꿀팁**을 전수합니다 (8번째 세미나 · 자료는 사내 온보딩 맥락을 다뤄 비공개). 프론티어 리서치 세미나(위)부터 신입 온보딩까지, 대상과 난이도를 아우르는 **폭넓은 강의 스펙트럼**입니다.
